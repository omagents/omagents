# OmAgents Icon Design

## Context
Create a project icon for the OmAgents OpenCode plugin.

## Chosen Concept
**AI Chip / Agent Node** - a rounded-square chip container with a robot head and an agent-node antenna.

## Rationale
- The square container gives a solid, app-icon presence while still referencing the "chip" / infrastructure layer.
- The robot head inside makes the AI/agent identity immediately recognizable.
- The antenna ends with a glowing cyan dot, representing the OmAgents "agent node".
- Flat and minimal style scales well from favicon to README display.

## Visual Spec
- **Container**: Rounded square with dark slate gradient (#0F172A -> #1E293B, rx=80).
- **Robot head**: Vibrant blue centered circle (#3B82F6, r=100).
- **Eyes**: Two soft-white circles (#E2E8F0, cx=226,286 / cy=246, r=14).
- **Mouth**: Soft-white rounded horizontal line (#E2E8F0).
- **Antenna**: Blue line rising from the head, capped by the agent dot.
- **Agent node**: Bright cyan circle (#22D3EE, r=18) at the antenna tip.
- **Colors**: Dark slate container -> vibrant blue head -> soft-white features -> bright cyan accent. Strong contrast, clear layering, modern feel.
- **Style**: Flat, minimal, robot/AI feel, square aspect ratio.
- **Canvas**: viewBox="0 0 512 512", square aspect ratio.
